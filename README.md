# joker-bot
An AI written in Python which tells random jokes

## Pre-requisites:

Python 3.9.2

## Building

### Install virtualenv

`pip3 install virtualenv`

### Create virtual environment

`virtualenv env`

### Activate virtual environment

`source env/bin/activate`

### Install requirements

`pip3 install -r requirements.txt`

## Running

`python3 main.py`
